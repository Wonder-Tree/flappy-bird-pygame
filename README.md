# Flappy Bird (Augmented Reality)

A clone of the popular app *Flappy Bird*, using Pygame and PoseCamera sdk

## Requirements

* Python 3.6

## How to run

Install dependencies
```
pip install -r requirements.txt
```

Run the game
```
python flappybird.py
```
